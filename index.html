工作坊特色

實務範例與練習

講義的內容與練習題都與實務相關，跳脫枯燥的語法，認識生活中的科技服務是如何用程式實作的。透過練習題動手，加深新知識觀念應用。

講師即時協助

擺脫線上課程一個人學習的孤獨及卡關無人可詢問的狀況，我們有經過扎實培訓的講師，在工作坊的過程中及時給予各種幫助。

AI 學習夥伴

學會運用 ChatGPT，成為你 24 小時不間斷的學習夥伴。讓你理解如何與 AI 對話，請它協助你引導你完成練習題、理解程式概念。

運作方式與理念

這份教材的初衷是為了幫助更多女性可以在 Python 社群中成為主動的參與者與領導者，讓講師透過培訓機制，不需自行編撰教材，可以專注在分享與成長。

我們很在乎入門活動的內容的品質， 因為對於完全沒有相關知識的初學者而言，講師講述的一言一語都有可能被奉為圭臬。 所以從教材準備到講授，我們都會很嚴格的把關，講者從面試經過培訓試教通過，才能講授入門工作坊。

常見問題 FAQ

現在 AI 工具越來越強大，確實能快速產出程式，但有個前提「你得要知道自己要什麼，並能判斷結果是否正確」。
這個工作坊除了學習 Python 基礎，讓你看得懂 AI 工具的產出內容，知道程式該怎麼修之外；
也透過大量的實務練習的解題過程，讓你知道怎麼一步步拆解需求，能更清楚對 AI 下指令。
讓你真的能「用 AI 寫程式」，不是「被 AI 寫程式」。

這個工作坊希望的是透過一個週末的時間，成為引領大家進入 Python 世界的角色。
所以比起學習更多的語法，我們安排體驗 Python 的各種應用，引起大家興趣建立未來更深入學習 Python 的動機。

工作坊其中一個特點就是講師能即時協助大家，線上的互動方式並不是那麼的良好，因此目前此工作坊都以線下實體活動為主，也沒有規劃錄影或直播等方式。
