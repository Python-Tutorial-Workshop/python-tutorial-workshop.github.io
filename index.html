運作方式

這份教材的初衷是為了幫助更多女性可以在 Python 社群中成為主動的參與者與領導者，
因此與台灣相關的 PyLadies 社群：PyLadies Kaohsiung, PyLadies Taiwan合作培訓不少講師。
也曾與美國在台協會-美國資料中心、WTM 女高中生資訊營合作舉辦不少場活動。

我們很在乎活動的品質，尤其是入門的相關內容，
因為對於完全沒有相關知識的初學者而言，講師講述的一言一語都有可能被奉為圭臬，

常見問題 FAQ

現在 Vibe Coding 正夯，為什麼還要從基礎開始學？

現在 AI 工具越來越強大，確實能快速產出程式，但有個前提
「你得要知道自己要什麼，並能判斷結果是否正確」。
這個工作坊除了學習 Python 基礎，讓你看得懂 AI 工具的產出內容，知道程式該怎麼修之外；
也透過大量的實務練習的解題過程，讓你知道怎麼一步步拆解需求，能更清楚對 AI 下指令。
讓你真的能「用 AI 寫程式」，不是「被 AI 寫程式」。

為什麼沒有函式、類別繼承、除錯等內容？

這個工作坊希望的是透過一個週末的時間，成為引領大家進入 Python 世界的角色。
所以比起學習更多的語法，我們安排體驗 Python 的各種應用，
引起大家興趣建立未來更深入學習 Python 的動機。

有沒有錄影或是線上參與的方式呢？

工作坊其中一個特點就是講師能即時協助大家，
線上的互動方式並不是那麼的良好，因此目前此工作坊都以線下實體活動為主，
也沒有規劃錄影或直播等方式。
